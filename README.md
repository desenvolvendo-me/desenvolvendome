# Informações Técnicas

## Variáveis de Ambiente

### Github
-  ENV['GITHUB_API_KEY']
-  ENV['GITHUB_CLIENT_ID']
-  ENV['GITHUB_CLIENT_SECRET']

## Running

> Clone o projeto

> Instale o ruby 2.6.5

> Agora execute

```
bundle install
rails db:create
rails db:migrate
rails db:seed 
rails s
```
